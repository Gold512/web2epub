# web2epub

Get data from webnovel sites and convert to epub

## Installation

Requirements 

- NodeJS
- npm
- git (easier download)


```bash
git clone https://github.com/Gold512/web2epub
```

Alternatively, download the repository

Then in the cloned directory

```bash
npm install
```
